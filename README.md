Installation and Running
1. Clone the repo
   https://github.com/FredrikMohag/workflow-CA.git

2. Install the dependencies
   npm install

3. Run build/SASS
   nmp run start
   npm build

4. Check format
   npm run format

5. Check Lint
   npm runt lint
   

Testing

Testing with Actions

Run Unit-test

npm run test:unit
